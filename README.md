# Functionality
- Loading recipes data from json-server
- Adding new recipe
- Adding recipe to favorites
- Searching by recipes

# Used technologies
-  React
-  TypeScript
-  Redux Toolkit
-  RTK Query
-  React Router Dom
-  JSON Server
-  React Icons
-  Vite
-  HTML
-  CSS

# How to run
Open 2 terminals:
- in first terminal paste ```json-server db.json --port 4200 --watch```
- in second terminal paste ```npm run dev```
